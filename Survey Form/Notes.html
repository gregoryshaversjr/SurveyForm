Notes Radio Button - Radio Buttons have inputs. The Radio button needs to be nested within a label. By wrapping an input button into a label it will automatically associate the radio button with the label. All related radio buttons should have the same "name" attribute to create a radio button group. This will insure one radio button is selected at one time. It's also best best practices to create an id attribute within the input tag and a for attribute within the label. This allows assistive technologies to create a link relationships.
